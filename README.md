# MediDiary: Breast Cancer Histopathology Image Analysis Using Deep Learning 🧠🔬

## 📌 Project Overview

**MediDiary** is an AI-powered web application developed as a final year engineering project aimed at automating the detection of breast cancer using histopathology images. The system enables patients to upload their medical images and provides doctors with reliable and fast diagnostic results using deep learning models.

---

## 🎯 Objective

To build a smart medical diagnostic system using image processing and machine learning algorithms to:
- Detect tumors from breast histopathology images
- Reduce manual errors in diagnosis
- Enhance early cancer detection and doctor-patient communication

## 🔍 Features

- 📤 Upload histopathology images (patients)
- 🤖 Analyze images using pre-trained CNN and SVM models
- 🧪 Image preprocessing using Otsu’s Thresholding & MRI techniques
- 🧾 Generate reports with prediction results (benign/malignant)
- 👨‍⚕️ Doctor access portal to view results and download reports
- 🔒 Secure and responsive user interface

---

## 🧠 Machine Learning Details

### Dataset
- 7,000+ labeled histopathology images (Benign & Malignant)
- Preprocessed using grayscale conversion, resizing, thresholding, etc.

### Algorithms Used
- **CNN (Convolutional Neural Network)** for image feature extraction and classification
- **SVM (Support Vector Machine)** for secondary classification validation
- Hybrid model increases accuracy and reduces false positives
